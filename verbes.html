🔎 Trouve l'infinitif

Voici une forme conjuguee. Quel est le verbe a l'infinitif ?

—

Score0/0 Serie0

Strategie

  • Reconnaitre la racine du verbe (fais- → faire, prend- → prendre).
  • Les terminaisons -ais / -ait / -ions indiquent l'imparfait.
  • Les terminaisons en -rai / -ras indiquent le futur.

✅ Participes passes

Quel est le participe passe de ce verbe ?

—

Score0/0 Serie0

Rappel

  • -er → -e : parler → parle, manger → mange
  • -ir → -i : finir → fini, choisir → choisi
  • Irreguliers : prendre → pris, voir → vu, faire → fait, mettre → mis

  • aller / venir — aller ↔ venir

  • monter / descendre
  • naitre / mourir
  • tomber / rester
  • retourner / passer
